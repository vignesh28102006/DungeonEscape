# Dungeon Escape 🏰

Dungeon Escape is a Java-based GUI game where the player explores a dungeon, encounters monsters, collects items, and tries to survive and escape.
This project was developed using Java Swing and demonstrates object-oriented programming and event-driven design.

---

## 🎮 Features

* 🧭 20x20 dungeon grid system
* ⚔️ Combat system (attack or flee)
* 👾 Random monster encounters
* 🎒 Item collection and usage
* 📊 Player stats (health, attack, defense)
* 🖥️ Interactive GUI built with Java Swing

---

## 🛠️ Tech Stack

* Java
* Java Swing (GUI Framework)
* Object-Oriented Programming (OOP)

---

## 📂 Project Structure

```
src/DungeonEscape/
│
├── DungeonEscapeMain.java   # Entry point of the application
├── GameController.java      # Handles game logic
├── GameFrame.java           # GUI interface using Swing
├── GameState.java           # Maintains game state
├── Player.java              # Player model and stats
├── Monster.java             # Monster logic and attributes
├── Item.java                # Item system
├── DungeonMap.java          # Map and grid handling
```

## ▶️ How to Run

1. Clone the repository:

   ```
   git clone [https://github.com/vignesh28102006/DungeonEscape]
   ```

2. Open the project in an IDE like Eclipse or IntelliJ

3. Navigate to:

   ```
   DungeonEscapeMain.java
   ```

4. Run the main method to start the game
